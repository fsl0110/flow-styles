# Color

Set a color to an element.

## Color Color

Every \$brand-colors entry is possible to use.

## Examples

#### Color Petrol: **co-petrol**

```html
<div class="co-petrol">Item</div>
```

```css
.co-petrol {
  color: #00979e;
}
```

#### Color Silver: **co-silver**

```html
<div class="co-silver">Item</div>
```

```css
.co-petrol {
  color: #d7d7d7;
}
```
