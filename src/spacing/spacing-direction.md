# Spacing Direction

## Types

- margin (m)
- padding (p)

## Options

- all ()
- vertical (v)
- horizontal (h)
- top (t)
- left (l)
- bottom (b)
- right (r)

## Sizes

Every \$size-multipliers are possible to use. Predefined sizes are: 0, 1, 2, 3, 4, 5, 6, 8, 16, 30

### Margin all directions value 2: **sp-m-2**

```html
<div class="sp-m-2">Item</div>
```

```css
.sp-m-2 {
  margin: 0.8rem;
}
```
