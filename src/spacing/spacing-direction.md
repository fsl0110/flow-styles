# Spacing Direction

## Types

- margin (m)
- padding (p)

## Options

- all ()
- vertical (v)
- horizontal (h)
- top (t)
- left (l)
- bottom (b)
- right (r)

## Sizes

Every \$size-multipliers are possible to use. Predefined sizes are: 0, 1, 2, 3, 4, 5, 6, 8, 16, 30

### Margin all directions: **sp-m2**

```html
<div class="sp-m2">Item</div>
```

```css
.sp-m2 {
  margin: 0.8rem;
}
```

### Padding all directions: **sp-p2**

```html
<div class="sp-p2">Item</div>
```

```css

  padding: 0.8rem;
}
```
