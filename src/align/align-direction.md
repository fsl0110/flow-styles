# Align Direction

Set flex-direction to a Flexbox-Container.

## Options

- horizontal (h)
- horizontal reverse (hr)
- vertical (v)
- vertical reverse (vr)

## Wrap Options

- wrap (w)
- wrap reverse (wr)

### Align Direction Horizontal: **al-dh**

```html
<div class="al-dh">Item</div>
```

### Align Direction Horizontal Reverse: **al-dhr**

```html
<div class="al-dhr">Item</div>
```
